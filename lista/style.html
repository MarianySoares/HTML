Desenvolvimento web

O que é desnvolvimento web?

O desenvolvimento web inclui criação, codificação e programação de sites e seus respectivos elementos e linguagens. Ele também envolve integração cpm sistemas e ferramentas,armazenamento de banco de dados, cibersegurança,entre outros, com o objetivo de estruturar cada detalhe de uma página da internet.

O que faz um desenvolvedor web?

O desenvolvedor de sites,chamado de profissional de Dev ou web Developer, é o reponsável por criar todo o planejamento estrutural de página utilizando linguagensde programação como HTML, CSS, JavaScript PHP, entre várias outras.
